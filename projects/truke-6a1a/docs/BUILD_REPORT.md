# Build report — Truke

## Bespoke app (generation swarm)

Truke es una aplicación estilo Tinder para intercambiar y regalar cosas usadas. Los usuarios pueden deslizar fotos de objetos, publicar los suyos, y chatear de forma anónima si hay un match mutuo sobre los items que hicieron match.

- Tablas: items, swipes, matches, messages
- Rutas API: app/api/items/create, app/api/feed, app/api/swipes, app/api/matches, app/api/messages, app/api/messages
- Pantallas: /app, /app/publicar, /app/matches, /app/matches/[matchId]
