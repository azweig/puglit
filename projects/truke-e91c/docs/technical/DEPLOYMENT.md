# Despliegue de Truke

## Entornos
- **Desarrollo**: Configuración local con variables de entorno específicas para desarrollo.
- **Producción**: Despliegue en Fly.io con configuración optimizada para producción.

## Migraciones
- Uso de herramientas de migración de base de datos para gestionar cambios en el esquema de PostgreSQL.

## Despliegue en Fly.io
- Configuración de la aplicación para ser desplegada en Fly.io, aprovechando su capacidad para aplicaciones distribuidas.

## Despliegue en Vercel
- Uso de Vercel para el despliegue del frontend, aprovechando su integración con Next.js.

## CI/CD
- Integración continua configurada para pruebas automáticas y despliegues en Fly.io y Vercel.

## Observabilidad
- Implementación de herramientas de monitoreo y logging para asegurar la salud y rendimiento de la aplicación.
