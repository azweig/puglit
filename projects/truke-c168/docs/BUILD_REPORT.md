# Build report — Truke

## Bespoke app (generation swarm)

Truke es una aplicación tipo Tinder para intercambiar y regalar cosas usadas. Los usuarios pueden deslizar fotos de objetos, publicar los suyos, y si hay un match mutuo, pueden chatear de forma anónima solo sobre los items que hicieron match.

- Tablas: items, swipes, matches, messages
- Rutas API: app/api/items/publish, app/api/feed, app/api/swipes, app/api/matches, app/api/messages, app/api/messages
- Pantallas: /app, /app/publicar, /app/matches, /app/chat/[matchId]
