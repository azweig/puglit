# Despliegue de Promiedos

## Entornos
- **Desarrollo**: Configurado localmente con variables de entorno específicas.
- **Producción**: Desplegado en Fly.io con configuración de entorno segura.

## Migraciones
- Uso de herramientas de migración de base de datos para aplicar cambios de esquema en PostgreSQL.

## Despliegue en Fly.io
- Configuración de `fly.toml` para definir la aplicación y sus servicios.
- Uso de `flyctl` para gestionar despliegues y escalado.

## Despliegue en Vercel
- Configuración de proyecto en Vercel para despliegue continuo del frontend.

## Integración Continua (CI)
- Configuración de pipelines de CI para pruebas automáticas y despliegue continuo.

## Observabilidad
- Implementación de monitoreo y logging para rastrear el rendimiento y errores de la aplicación.
