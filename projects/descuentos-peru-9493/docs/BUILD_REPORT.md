# Build report — Descuentos Perú

## Bespoke app (generation swarm)

Descuentos Perú es una aplicación web que permite a los usuarios ver qué restaurantes y tiendas cercanas ofrecen descuentos basados en los programas de lealtad que poseen y su ubicación actual.

- Tablas: programs, user_memberships, merchants, branches, offers
- Rutas API: app/api/programs, app/api/user_memberships, app/api/user_memberships, app/api/location, app/api/nearby_offers, app/api/merchants, app/api/branches, app/api/offers
- Pantallas: /app, /app/memberships, /app/location, /app/merchants, /app/branches, /app/offers
