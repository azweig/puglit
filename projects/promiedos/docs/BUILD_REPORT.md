# Build report — Promiedos

## Bespoke app (generation swarm)

Promiedos clone for live Argentine football: view live matches, fixtures, standings, top scorers, and historical results. Data is scraped and updated via cron jobs.

- Tablas: matches, tournaments, standings, goal_scorers
- Rutas API: app/api/v1/live-football-data, app/api/v1/matches, app/api/v1/tournaments, app/api/v1/standings, app/api/v1/goal-scorers
- Pantallas: /app, /app/matches/create, /app/tournaments/create, /app/standings/create, /app/goal-scorers/create
