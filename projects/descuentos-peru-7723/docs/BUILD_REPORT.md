# Build report — Descuentos Perú

## Bespoke app (generation swarm)

Descuentos Perú es una aplicación web que permite a los usuarios ver descuentos en restaurantes y tiendas cercanas basados en sus programas de lealtad y ubicación actual. Los usuarios pueden gestionar sus programas de lealtad, establecer su ubicación y ver ofertas cercanas que aplican a sus membresías.

- Tablas: programs, user_memberships, merchants, branches, offers
- Rutas API: app/api/programs.ts, app/api/user/memberships.ts, app/api/user/memberships.ts, app/api/user/location.ts, app/api/nearby/offers.ts, app/api/merchants.ts, app/api/branches.ts, app/api/offers.ts
- Pantallas: /app, /app/programs, /app/location, /app/merchants, /app/branches, /app/offers
