# Build report — Descuentos Perú

## Bespoke app (generation swarm)

Descuentos Perú es una aplicación web que permite a los usuarios descubrir descuentos en restaurantes y tiendas cercanas basados en sus programas de lealtad y ubicación actual. Los usuarios pueden seleccionar sus programas de lealtad, guardar su ubicación y ver ofertas cercanas que aplican a sus membresías.

- Tablas: programs, user_memberships, merchants, branches, offers
- Rutas API: app/api/programs/route.ts, app/api/user_memberships/route.ts, app/api/user_memberships/route.ts, app/api/location/route.ts, app/api/discounts/route.ts, app/api/merchants/route.ts, app/api/branches/route.ts, app/api/offers/route.ts
- Pantallas: /app, /app/memberships, /app/location, /app/merchants, /app/branches, /app/offers
