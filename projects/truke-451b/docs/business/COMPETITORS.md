# Análisis de Competidores para Truke

## Competidores Reales/Probables
1. **Wallapop**: Plataforma de compra-venta de segunda mano.
2. **Letgo**: Aplicación para vender y comprar artículos usados.
3. **Facebook Marketplace**: Espacio dentro de Facebook para intercambio y venta de objetos.

## Posicionamiento
- **Truke** se posiciona como una plataforma de intercambio gratuito y anónimo, enfocándose en el sistema de coincidencias para facilitar el intercambio de objetos.

## Diferenciación
- **Interfaz tipo Tinder**: Única en el sentido de utilizar un sistema de coincidencias para el intercambio.
- **Anonimato en el chat**: Aumenta la privacidad y seguridad de los usuarios.

## Comparación de Características
| Característica              | Truke   | Wallapop | Letgo | Facebook Marketplace |
|----------------------------|---------|----------|-------|----------------------|
| Intercambio Anónimo        | Sí      | No       | No    | No                   |
| Sistema de Coincidencias   | Sí      | No       | No    | No                   |
| Gratuito                   | Sí      | Sí       | Sí    | Sí                   |
| Funciones Premium          | No      | No       | No    | No                   |
| Base de Usuarios Amplia    | No      | Sí       | Sí    | Sí                   |
