# Mesa

Generated by [Puglit](https://puglit.com) — a complete, deployable SaaS spine configured for **Mesa**.

## What's inside
- **Next.js 16** app (App Router) — branded landing + auth pages, all driven by `domain.config.ts`
- **Auth**: JWT (web cookie + mobile Bearer), magic links, email verification, password reset
- **Postgres** data layer, **analytics**, **rate-limit middleware**
- **infra/**: Dockerfile + Fly.io template · **sql/**: schema to run in your database
- Edit `domain.config.ts` to change branding, plans, modules — the whole app follows.

## Run it
```bash
npm install
# set env (see infra/secrets.example.env): POSTGRES_*, JWT_SECRET, APP_URL, …
# run the SQL in sql/*.sql against your Postgres
npm run dev
```

**Modules enabled:** growth, payments, emailLifecycle.
