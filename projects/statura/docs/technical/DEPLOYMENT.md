# Despliegue de Statura

## Entornos
- **Desarrollo**: Local con Next.js y PostgreSQL.
- **Producción**: Desplegado en Fly.io.

## Migraciones
- Utilizar herramientas de migración de base de datos para aplicar cambios de esquema.

## Despliegue en Fly.io
- Configurar Fly.io para el despliegue continuo de la aplicación.
- Asegurar que las variables de entorno estén configuradas correctamente para producción.

## CI/CD
- Integrar con un sistema CI para pruebas automáticas y despliegue continuo.

## Observabilidad
- Implementar monitoreo y logging para detectar y resolver problemas rápidamente.
- Utilizar herramientas como Prometheus o Grafana para la visualización de métricas.
