# Competencia de Truke

## Competidores Reales/Likely
1. **Wallapop**: Plataforma para comprar y vender productos de segunda mano.
2. **Letgo**: Aplicación para vender cosas usadas localmente.
3. **Facebook Marketplace**: Funcionalidad de Facebook para comprar y vender objetos usados.

## Posicionamiento
- Truke se posiciona como la app para intercambiar objetos de forma anónima y gratuita, enfocada en la experiencia de usuario tipo "deslizamiento".

## Diferenciación
- **Anónimo**: A diferencia de otras plataformas, Truke permite chats anónimos.
- **Intercambio Directo**: Enfocada en el intercambio y regalo, no en la venta.

## Comparación de Características
| Característica         | Truke | Wallapop | Letgo | Facebook Marketplace |
|------------------------|-------|----------|-------|----------------------|
| Interfaz de Deslizamiento | Sí    | No       | No    | No                   |
| Chat Anónimo          | Sí    | No       | No    | No                   |
| Enfoque en Intercambio | Sí    | No       | No    | No                   |
| Gratuita              | Sí    | Sí       | Sí    | Sí                   |
