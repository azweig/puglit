# Build report — Promiedos

## Bespoke app (generation swarm)

Promiedos clone provides real-time updates on Argentine football, including live match scores, fixtures, league standings, top scorers, and historical results. Data is scraped and updated regularly, offering users a comprehensive view of ongoing and upcoming matches.

- Tablas: matches, leagues, players, fixtures
- Rutas API: app/api/live-matches/route.ts, app/api/matches/route.ts, app/api/leagues/route.ts, app/api/players/route.ts, app/api/fixtures/route.ts
- Pantallas: /app, /app/matches/create, /app/leagues/create, /app/players/create, /app/fixtures/create
