# Despliegue de Truke

## Entornos
- **Desarrollo**: Configuración local con variables de entorno específicas.
- **Producción**: Desplegado en Fly.io con configuración optimizada para rendimiento y seguridad.

## Migraciones
- Uso de herramientas de migración para gestionar cambios en el esquema de la base de datos PostgreSQL.

## Despliegue en Fly.io/Vercel
- **Fly.io**: Despliegue del backend y base de datos.
- **Vercel**: Despliegue del frontend.

## CI/CD
- Integración continua configurada para ejecutar pruebas y desplegar automáticamente en cada commit a la rama principal.

## Observabilidad
- Monitoreo de logs y métricas de rendimiento a través de herramientas integradas en Fly.io y Vercel.
