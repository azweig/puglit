# Build report — Noteflow

## Bespoke app (generation swarm)

Noteflow es una aplicación de notas que permite a los usuarios crear, gestionar y organizar notas con recordatorios y etiquetas. Los usuarios pueden añadir recordatorios a sus notas para recibir alertas en momentos específicos y categorizar sus notas con etiquetas de colores para una mejor organización.

- Tablas: notes, reminders, tags, note_tags
- Rutas API: app/api/notes/route.ts, app/api/notes/route.ts
- Pantallas: /app, /app/note/new
