# Build report — Truke

## Bespoke app (generation swarm)

Truke es una aplicación para intercambiar y regalar cosas usadas, donde los usuarios pueden deslizar fotos de objetos para indicar interés, publicar sus propios objetos, y chatear anónimamente si hay un match mutuo en los objetos.

- Tablas: items, swipes, matches, messages
- Rutas API: app/api/feed/route.ts, app/api/swipes/route.ts, app/api/matches/route.ts, app/api/messages/route.ts, app/api/messages/route.ts
- Pantallas: /app, /app/matches, /app/chat/[matchId]
