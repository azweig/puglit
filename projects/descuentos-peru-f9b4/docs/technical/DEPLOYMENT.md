# Despliegue de Descuentos Perú

## Entornos
- **Desarrollo**: Entorno local para pruebas y desarrollo.
- **Producción**: Desplegado en Fly.io para acceso público.

## Migraciones
- Uso de herramientas de migración de bases de datos para aplicar cambios en el esquema de PostgreSQL de manera controlada.

## Despliegue en Fly.io
- Configuración de Fly.io para el despliegue continuo de la aplicación.
- Uso de Docker para empaquetar la aplicación y sus dependencias.

## Despliegue en Vercel
- Uso de Vercel para el despliegue del frontend de Next.js.

## Integración Continua (CI)
- Configuración de pipelines de CI para automatizar pruebas y despliegues.

## Observabilidad
- Implementación de herramientas de monitoreo y logging para asegurar la disponibilidad y el rendimiento de la aplicación.
- Uso de servicios de alertas para notificar sobre problemas de rendimiento o errores críticos.
