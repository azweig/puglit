# Deployment

## Environments
- **Development**: Local environment for development.
- **Staging**: Pre-production environment for testing.
- **Production**: Live environment for end-users.

## Migrations
- Database migrations are handled using a migration tool compatible with PostgreSQL.

## Deployment Process
- **Fly.io**: Used for deploying the backend services.
- **Vercel**: Used for deploying the frontend (Next.js).

## Continuous Integration (CI)
- Automated tests and builds are triggered on each commit to the main branch.

## Observability
- **Logging**: Centralized logging for monitoring application health.
- **Monitoring**: Use of monitoring tools to track application performance and uptime.
