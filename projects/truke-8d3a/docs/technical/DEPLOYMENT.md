# Despliegue de Truke

## Entornos
- **Desarrollo**: Configurado para pruebas locales y desarrollo continuo.
- **Producción**: Configurado para despliegue en Fly.io con optimizaciones de rendimiento.

## Migraciones
- Uso de herramientas de migración de base de datos para gestionar cambios en el esquema de PostgreSQL.

## Despliegue en Fly.io
- Configuración de Fly.io para despliegue continuo del backend.
- Uso de Docker para empaquetar la aplicación.

## Despliegue en Vercel
- Configuración de Vercel para el despliegue continuo del frontend construido con Next.js.

## Integración Continua (CI)
- Configuración de pipelines de CI para pruebas automáticas y despliegue.

## Observabilidad
- Implementación de herramientas de monitoreo y logging para asegurar la estabilidad y rendimiento de la aplicación en producción.
