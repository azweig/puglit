# Build report — Descuentos Perú

## Bespoke app (generation swarm)

Descuentos Perú es una aplicación web que permite a los usuarios ver ofertas y descuentos en tiendas y restaurantes cercanos basados en los programas de lealtad que poseen y su ubicación actual. Los usuarios pueden gestionar sus programas de lealtad y actualizar su ubicación para recibir recomendaciones personalizadas.

- Tablas: programs, user_memberships, merchants, branches, offers
- Rutas API: app/api/programs, app/api/user/memberships, app/api/user/memberships, app/api/user/location, app/api/offers/nearby, app/api/merchants, app/api/branches, app/api/offers
- Pantallas: /app, /app/programs, /app/location, /app/merchants, /app/branches, /app/offers
