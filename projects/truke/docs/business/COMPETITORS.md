# Análisis de Competencia para Truke

## Competidores Directos
- **Wallapop:** Plataforma líder en España para compra-venta de segunda mano.
- **Letgo:** Aplicación similar que permite la compra-venta de objetos usados.

## Competidores Indirectos
- **Facebook Marketplace:** Amplia base de usuarios y facilidad de uso.
- **eBay:** Plataforma global de subastas y ventas de segunda mano.

## Posicionamiento
- **Truke:** Se posiciona como una app de intercambio, no de compra-venta, con un enfoque en la facilidad de uso y el anonimato.

## Diferenciación
- **Anonimato en el Chat:** A diferencia de otras plataformas, Truke ofrece un chat anónimo.
- **Modelo de "Match":** Basado en deslizamiento, similar a Tinder, para facilitar el proceso de intercambio.

|------------------------|-------|----------|-------|----------------------|------|
