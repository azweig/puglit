# spine/ — the SaaS skeleton

This is where the opinionated, domain-agnostic Next.js 15 spine is extracted from the production TodoAstros codebase (cleaned of domain coupling, templatized, toggle-driven).

Being built slice by slice — see [`../modules/README.md`](../modules/README.md) for per-module status and extraction order. Slice 2 (Core: app-shell, db, auth, middleware, analytics, admin, observability) is next.

Nothing here is generated by an LLM at runtime — the spine is assembled deterministically. The LLM only ever writes the **domain** (declared in `../domain.config.ts`).
